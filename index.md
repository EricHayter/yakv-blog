## Features

- **Deep Black Design** - Pure black background with comfortable contrast
- **Technical Typography** - Geist Mono font throughout for a consistent look
- **Minimal Aesthetic** - Clean borders and reduced visual clutter
- **Optimized Readability** - Carefully chosen colors to reduce eye strain
- **Built-in Search** - Fast, client-side search
- **Responsive** - Mobile-first design that works everywhere

## Getting started

This theme is built on [Just the Docs](https://just-the-docs.com) with a complete visual overhaul focused on a minimal, technical aesthetic.

### Quick start

1. Clone or fork this repository
2. Install dependencies with `bundle install`
3. Run `bundle exec jekyll serve`
4. Visit `http://localhost:4000`

### Customization

The theme uses the `techno` color scheme defined in `_sass/color_schemes/techno.scss`. You can easily customize:

- Colors and backgrounds
- Typography and font sizes
- Spacing and borders
- Button styles and more

See the [README](https://github.com/yourusername/technoblog/blob/main/README.md) for detailed customization options.

## About

Built as a fork of Just the Docs by Patrick Marsceill, this theme maintains the excellent navigation and search features while providing a completely new visual design focused on:

1. **Minimal distraction** - Dark mode optimized for long reading sessions
2. **Technical aesthetic** - Monospace fonts and clean lines
3. **Content first** - Simple design that makes your writing shine
4. **Easy on the eyes** - Reduced contrast and carefully chosen colors
